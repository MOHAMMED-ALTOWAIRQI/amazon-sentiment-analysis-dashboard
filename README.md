# Amazon Sentiment Analysis Dashboard

## Overview
This project is an AI-based sentiment analysis system designed to classify Amazon product reviews into positive or negative sentiments. 
The system processes large-scale review data and provides insights through an interactive dashboard.

## Features
- Sentiment classification using Machine Learning
- Text preprocessing and TF-IDF vectorization
- Logistic Regression model for prediction
- Interactive GUI built with Tkinter
- Data visualization (bar charts, pie charts, histograms)
- Product-level analysis and insights

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Tkinter

## How It Works
1. Load dataset (CSV file)
2. Preprocess text data
3. Apply TF-IDF vectorization
4. Train or use trained ML model
5. Predict sentiment (Positive / Negative)
6. Display results and charts in dashboard

## Dataset
The system uses a dataset of Amazon product reviews containing text, ratings, and product information.
